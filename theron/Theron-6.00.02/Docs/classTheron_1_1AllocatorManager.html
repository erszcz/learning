Detailed Description

Static class that manages allocators for use by Theron.

Static SetAllocator and GetAllocator methods on the class allow the allocator used by Theron to be set and retrieved. Setting the allocator replaces the DefaultAllocator, which is used if no custom allocator is explicitly set. The GetAllocator method returns a pointer to the currently set allocator, which is either the allocator set previously using SetAllocator or an instance of DefaultAllocator, if none has been set.

class MyAllocator : public Theron::IAllocator
{
public:
MyAllocator();
virtual ~MyAllocator();
virtual void *Allocate(const SizeType size);
virtual void *AllocateAligned(const SizeType size, const SizeType alignment);
virtual void Free(void *const memory);
virtual void Free(void *const memory, const SizeType size);
};
MyAllocator allocator;
Theron::AllocatorManager::SetAllocator(&allocator);
Note
The SetAllocator method can be called at most once, at application start. If the DefaultAllocator is replaced with a custom allocator then it must be replaced at application start, before any Theron objects (endpoints, frameworks, actors or receivers) are constructed. GetAllocator can be called any number of times after SetAllocator is called.
