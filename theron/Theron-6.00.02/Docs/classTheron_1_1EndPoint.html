Detailed Description

Network communication object.

Enables communication with actors in other processes and on remote hosts. A single instance of this class must be constructed in each application that is intended to be used in a collection of distributed applications in distributed computing.

If an EndPoint is created then it must outlive any frameworks, actors and receivers that are tied to it.
